Define diseño interactivo usando un dibujo sin palabras. Incluye tu dibujo en el documento.

Contesta la siguiente pregunta: ¿Para hacer un sitio web se requiere saber de diseño interactivo? ¿Por qué?

Yo creo que si se necesita, porque aunque tengas conocimiento de cómo programar, eso no significa que tu página sea atractiva e intuitiva para el usuario. Pienso que el diseño interactivo le facilita el camino al usuario para que la página web cumpla el objetivio con el que fue creado.

¿Qué otros productos requieren de conocer sobre diseño interactivo?

Aplicaciones móviles, cajeros, plataformas de streaming, sistemas operativos de celulares, computadoras y pantallas de televisión inteligentes, carros que cuenten con una pantalla táctil para interactuar, por ejemplo Tesla, entre otros productos.
 
Ejercicio

¿Para qué sirve el body y el head en un documento de html?

El body es una etiqueta que indica todo lo que se va a ver dentro de la página.

El head funciona como una etiqueta de metadata y contiene información de la página como: titulo, css y scripts.

Incluir la etiqueta title en el head de la página más fea del mundo.
